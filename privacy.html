Integritetspolicy

Senast uppdaterad: maj 2026

Viktor Andersson ("vi", "oss") värnar om din integritet. Denna policy beskriver hur Rädda kylen samlar in, använder och lagrar dina personuppgifter.

Data vi samlar in

  • E-postadress — används för att skapa och hantera ditt konto.
  • Ingredienser och skafferidata — sparas för att visa receptförslag anpassade efter vad du har hemma.
  • Inköpslista — sparas så att du kan komma åt den från appen.
  • Användarbeteende — anonymiserad statistik om hur appen används, t.ex. vilka recept som öppnas och hur ofta appen startas. Används för att förbättra appen.
  • Prenumerationsstatus — sparas för att hantera din Premium-prenumeration.

Hur vi använder din data

Din data används enbart för att appen ska fungera. Vi säljer aldrig din data till tredje part och använder den inte för marknadsföring.

Lagring

Din data lagras säkert hos Supabase (supabase.com), en EU-kompatibel databastjänst. Data lagras så länge ditt konto är aktivt.

Dina rättigheter

Du har rätt att:

  • Begära ut den data vi har om dig
  • Begära att din data rättas
  • Begära att ditt konto och all din data raderas

För att radera ditt konto, kontakta oss på e-postadressen nedan så hanterar vi din begäran inom 30 dagar.
